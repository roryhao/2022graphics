## 2022graphics
# 把修改的資料，推送到雲端
    4-1. cd 2022graphics 進入你的倉庫
    4-2. git status 看你的狀況（红的，多了目錄）
    4-3. git add .把檔案加進帳冊
    4-4. git status 再看一次（綠的）這裡要加入
            git config --global user. email "你的email"
            git config --global user.name“你的名”
            git commit -m “上傳第1週”
    4-5. git push 推送到雲端，BUT要登入
    4-6 在Chrome登陸GitHub
